# **App Name**: FuegoRegistro

## Core Features:

- Firefighter Management: Manage a roster of firefighters, including details like ID number, name, rank, firehouse, and active/inactive status.  Admins can add, edit, and deactivate firefighters.
- Session Creation: Create training sessions with details such as date, start time, specialization, title, and description. Add firefighters as instructors, assistants, or attendees.
- Attendance Tracking: Record attendance for each training session, filterable by firehouse and rank.  Track attendees, absences, excused absences, and tardiness, with summary counts at the top.
- Leave Management: Record firefighter leaves with start and end dates, and license type (ordinary, extraordinary, sanction, illness, study, etc.). Absences should automatically be updated except for sanctions.
- Reporting: Generate attendance reports filterable by date, firefighter, firehouse, groups/officers, session, and specialty.
- User Administration: Manage user accounts with roles: Administrator (full access), Operator (create sessions, track attendance, reports), Assistant (manage leaves, reports).

## Style Guidelines:

- Primary color:  Vivid red (#FF4136) to evoke urgency, bravery, and the core symbol of firefighting.
- Background color:  Light gray (#F0F0F0), a desaturated version of the primary red, creating a calm, neutral backdrop.
- Accent color:  Deep orange (#FF851B), an analogous color to red, used for call-to-action elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and 'Inter' (sans-serif) for body text.
- The application will have a responsive layout, ensuring it's accessible and easy to use on both desktop and mobile devices.
